## Job Market Analyzer

## what is this Project about?
An AI powered tool that analyzes job market for Data Engineers in 2026.

## Features
- Fetches real job listing data
- Analyzes most in demand skills
- Generates Insights using AI
- Visual Dashboards for Trends

- AI Integration
