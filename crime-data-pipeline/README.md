# Crime Data Pipeline Project

## What is this ?
An end-to-end modern data pipeline built on real India Crime Data (NCRB).
From Raw CSV -> Python -> Azure -> SQL -> Power BI Dashboard.

## Tech Stack
| Layer |  Tool |
|-------|-------|
| Ingestion | Python |
| Storage | Azure Blob Storage |
| Transform | Python + SQL + dbt |
| Visualize | Power BI |

## Data Source
NCRB (National Crime Records Bureau) — data.gov.in

In Progress — Building Phase 1: Data Ingestion
